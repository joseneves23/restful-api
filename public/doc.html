Documentação da API

As rotas disponíveis são:

Método Rota Descrição
GET / Retorna uma página de demonstração da API.
GET /about Retorna uma página com informações sobre os autores.
GET /doc Retorna uma página com documentação da API.
GET /students Retorna todos os estudantes.
GET /students/{id} Retorna um estudante específico pelo id.
POST /students Adiciona um novo estudante.
PUT /students/{id} Atualiza os dados de um estudante pelo id.
DELETE /students/{id} Remove um estudante específico pelo id.
